Візуалізація паркетності

Два способи побачити одні й ті самі дані. Натискайте на елементи для деталей.

Bubble Map: рубрики × місяці × паркетність
Розмір кола = кількість статей. Колір = % паркету (світлий → темний). Натисніть для деталей.
% паркету:
0% → 15%+ П1 П2
Heatmap: % паркету по місяцях та рубриках
Рядки = рубрики, колонки = місяці. Темніший колір = більше паркету. Натисніть для деталей.
% паркету:
0% → 20%+
