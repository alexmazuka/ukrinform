# CensorZero: Незалежний аудит оцінок ІМІ щодо Укрінформу

> **51 667 статей. Повний масив. Кожна стаття — відкрита для перевірки.**

## Що це?

Це незалежне дослідження рішення [Інституту масової інформації (ІМІ)](https://imi.org.ua) щодо виключення Укрінформу з [Білого списку](https://imi.org.ua/monitorings) у квітні 2024 року.

ІМІ оцінив ~100 статей за 2 дні. Ми зібрали **весь публічний масив** за обидва досліджувані періоди і зробили аналіз повністю **відтворюваним** — кожен може перевірити кожну статтю.

**🌐 Дашборд:** [alexmazuka.github.io/ukrinform](https://alexmazuka.github.io/ukrinform/)

## Методологія

Детальний опис: [`METHODOLOGY.md`](METHODOLOGY.md)

### Що ми перевіряємо

ІМІ назвав причиною виключення «паркетні повідомлення» та «недостатній баланс думок». Наш алгоритм перевіряє ці ж ознаки:

- **Ймовірний паркет:** офіційне джерело + ≤1 джерело + 0 неофіційних джерел
- **Ризик дисбалансу:** офіційне джерело + 0 неофіційних джерел + ≤1 джерело
- **Підрахунок джерел:** regex-пошук маркерів цитування у тексті статті
- **Класифікація джерел:** офіційні (Президент, Кабмін, Парламент, міністерства тощо) vs неофіційні

### Обмеження

Наш алгоритмічний аналіз — це **proxy-метрика**, не точна копія методології ІМІ. Алгоритм може давати хибнопозитивні та хибнонегативні результати. Саме тому кожна стаття доступна за прямим посиланням — перевірте самостійно.
